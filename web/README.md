# Frontend - UniHub Workshop

Chào mừng các bạn đến với phần Frontend của dự án UniHub Workshop! 🚀

Dưới đây là giải thích về cấu trúc thư mục cơ bản để mọi người trong team dễ dàng nắm bắt:

- **`components/`**: Chứa các thành phần UI dùng chung trong toàn bộ ứng dụng (ví dụ: Header, Footer, Button, CustomInput,...).
- **`pages/`**: Chứa các trang giao diện chính (ví dụ: AdminDashboard, StudentHome, LoginPage,...).
- **`services/`**: Chứa các file xử lý logic gọi API xuống Backend (ví dụ: authService.js, workshopService.js,...).
- **`assets/`**: Chứa các tài nguyên tĩnh như hình ảnh, CSS, fonts,...

Chúc team Frontend code thật mượt và không có bug nhé! ✨
